# Vortex – Personal Portfolio Website

**Live Demo:** https://vortex-portfolio.vercel.app  
**GitHub Repository:** https://github.com/Vortex-macos/vortex-portfolio

![Preview](https://vaxerski.xyz/github/TopGif.gif)

A futuristic, space-themed, cyberpunk-inspired portfolio built with **pure HTML, CSS, JavaScript + Three.js** — no frameworks, no build tools, just raw performance and beauty.

Deployed instantly on **Vercel** — blazing fast, zero config.

---

### Features

- Interactive 3D starfield background with mouse parallax (Three.js)
- Smooth scrolling + active navbar highlighting
- GSAP-powered scroll-triggered reveal animations
- Custom animated cursor with glowing trail
- Large, bold About section with holographic logo glow
- Animated GitHub & Gmail social icons
- Live typing SVG animation (just like my GitHub README)
- Fully responsive – looks perfect on mobile, tablet, and desktop
- 100% static – works offline and deploys anywhere

---

### Tech Stack

HTML5 • CSS3 • JavaScript • Three.js • GSAP • Font Awesome • Vercel```
